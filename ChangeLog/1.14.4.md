## [1.14.4] - 2026-01-04

### Fixed
- **CRITICAL: CSV Schema Conflict** (IMPROVEMENTS.md High Priority #1)
  - Fixed incompatible CSV schemas when multiple action types run on the same day
  - Each action type now creates its own log file: `_msgraphtool_{action}_{date}.csv`
  - Prevents data corruption and column misalignment in CSV logs
  - Examples:
    - `sendmail` → `_msgraphtool_sendmail_2026-01-03.csv`
    - `getevents` → `_msgraphtool_getevents_2026-01-03.csv`
    - `sendinvite` → `_msgraphtool_sendinvite_2026-01-03.csv`
    - `getinbox` → `_msgraphtool_getinbox_2026-01-03.csv`
- **Missing closing parenthesis in error message** (IMPROVEMENTS.md High Priority #2)
  - Fixed typo in authentication error message (src/msgraphtool.go:334)
  - Added missing closing parenthesis to error text

### Changed
- Updated CSV logging to create action-specific log files
- Updated documentation (CLAUDE.md, README.md) with new CSV filename format
- CSV log filenames now include action type for better organization and schema consistency
