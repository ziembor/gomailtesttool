# Version 2.6.18

## Changes

### Code Organization

Refactored SMTP tool TLS display functionality for improved code organization and maintainability.

**Added:**
- `cmd/smtptool/tls_display.go` - New file containing TLS display and formatting functions

**Features:**
- `displayComprehensiveTLSInfo()` - Console output for TLS connection and certificate details
- `displayTLSConnectionInfo()` - TLS connection information formatting
- `displayCertificateInfo()` - Certificate details formatting with warnings
- `formatTLSInfoForCSV()` - CSV export formatting for TLS data
- `TLSCSVData` struct for structured CSV output

### Benefits

- Improved code organization by separating display logic from main tool logic
- Better maintainability with dedicated file for TLS display functions
- Consistent TLS information display across all SMTP tool actions
- Reduced code duplication in sendmail.go, testconnect.go, and testauth.go
