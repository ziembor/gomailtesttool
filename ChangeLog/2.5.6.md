# Change Log: Version 2.5.6

**Release Date:** 2026-01-31

This release renames the Microsoft Graph tool binary from `msgraphgolangtestingtool` to `msgraphtool` for brevity.

## Changes

### Binary Rename

- **Renamed `msgraphgolangtestingtool` to `msgraphtool`** - The Microsoft Graph tool binary has been renamed for easier usage:
  - Windows: `msgraphtool.exe` (was `msgraphgolangtestingtool.exe`)
  - Linux: `msgraphtool` (was `msgraphgolangtestingtool`)
  - macOS: `msgraphtool` (was `msgraphgolangtestingtool`)

- **Updated release archive names**:
  - `msgraphtool-windows.zip`
  - `msgraphtool-linux-amd64.zip`
  - `msgraphtool-macos-arm64.zip`

- **Updated Go module name** from `msgraphgolangtestingtool` to `msgraphtool`

## Upgrade Notes

**Breaking Change**: The binary name has changed. Update any scripts or shortcuts that reference the old name:

```bash
# Old
./msgraphgolangtestingtool -action getinbox ...

# New
./msgraphtool -action getinbox ...
```
