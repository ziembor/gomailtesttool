# Change Log: Version 2.5.5

**Release Date:** 2026-01-31

This patch release specifies explicit architectures for Linux and macOS builds.

## Changes

### CI/CD Improvements

- **Linux builds now target amd64** - Linux binaries are built specifically for x86_64 architecture
- **macOS builds now target arm64** - macOS binaries are built specifically for Apple Silicon (M1/M2/M3)
- **Updated release asset naming** - Archives now include architecture in filename:
  - `msgraphtool-linux-amd64.zip`
  - `msgraphtool-macos-arm64.zip`

---

## Files Modified

- `.github/workflows/build.yml`
  - Added GOARCH environment variable for Linux (amd64) and macOS (arm64) builds
  - Updated artifact and zip names to include architecture
  - Updated release notes to reflect architecture-specific downloads

---

## Upgrade Notes

This is a CI/CD-only change. Users should download the appropriate archive for their platform:
- **Windows**: `msgraphtool-windows.zip`
- **Linux x86_64**: `msgraphtool-linux-amd64.zip`
- **macOS Apple Silicon**: `msgraphtool-macos-arm64.zip`
